# Support Ticket Management System вЂ” MERN Stack

## Overview

A full-stack MERN (MongoDB, Express.js, React, Node.js) web application that solves the **drawbacks of current ticket management systems**: complexity, high cost, poor UX, lack of real-time updates, missing personalization, data silos, and steep learning curves.

The system is designed for small-to-medium organizations and colleges with a **premium, modern dark-mode UI** inspired by ManageEngine ServiceDesk Plus and Freshdesk.

---

## Problems Solved (Drawbacks of Current Systems)

| Current System Drawback | Our Solution |
|---|---|
| Steep learning curve & complex UI | Clean, intuitive single-page dashboard |
| Expensive SaaS pricing | Free, self-hosted open-source MERN app |
| No real-time status updates | Socket.IO live notifications & real-time ticket feed |
| Generic automated responses | Personalized agent notes & reply templates |
| Data silos across tools | Unified dashboard: tickets, users, reports in one place |
| Poor mobile responsiveness | Fully responsive layout |
| No SLA tracking | Built-in SLA timer per ticket with color alerts |
| No knowledge base / self-service | Integrated FAQ/Knowledge Base section |
| Manual ticket assignment | Smart auto-assignment with workload balancing |
| Limited analytics | Live charts: resolution time, category breakdown, agent performance |
| No file attachment support | File/image upload on tickets |
| No audit trail | Full ticket activity log (timeline) |

## Key Features

### User Features
- Register / Login (JWT auth with refresh tokens)
- Create support tickets with category, priority, description, file attachment
- Real-time ticket status updates via Socket.IO
- SLA countdown timer per ticket
- Comment/reply thread on each ticket
- Feedback/rating after resolution
- Profile management with avatar upload
- Notification bell with real-time alerts
- Self-service Knowledge Base search

### Agent Features
- View assigned tickets with workload indicator
- Update ticket status: Open в†’ In Progress в†’ Resolved в†’ Closed
- Internal notes (not visible to users)
- Respond to users directly from ticket thread
- View own performance stats

### Admin Features
- Full dashboard: live stats, charts (Recharts)
- Assign/reassign tickets to agents
- Smart auto-assign (round-robin balancing)
- User management (activate/deactivate, change roles)
- Category & priority management
- SLA policy configuration
- Reports: resolution time, category breakdown, agent performance, CSAT
- Export reports as CSV/PDF
- Audit log of all ticket changes
